# Dotfiles
My dotfiles
## Tools
- **Sway:** Window manager
- **Waybar:** Top bar that displays some information like workspaces, battery, date..
- **Wofi:** Applications launcher
- **Foot:** Terminal emulator
- **Dunst:** Notification manager
- **Neovim:** Text editor
- **Zathura:** A simple PDF viewer with vim keybindings
